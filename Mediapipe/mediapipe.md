> # MediaPipe
> ------------------------------------------------------------------------------------------
>### 1. MediaPipe란?
>
> Google에서 개발한 실시간 멀티모달 인식 파이프라인 프레임워크
> 
> 영상, 이미지, 오디오 등의 입력 데이터를 기반으로 사람의 얼굴, 손, 포즈, 제스처 등을 고속·고정밀로 추론 할 수 있도록 설계되어있음
>
>
>
> * MediaPipe의 목적
>   * 실시간 처리에 최적화된 추론 파이프라인 제공
>   * 모바일, 임베디드, 엣지 환경에서도 안정적인 동작
>   * 딥러닝 기반 인식 모델을 쉽게 활용할 수 있는 고수준 API 제공
>
> ### 2. MediaPipe의 핵심 특징
>
> * 실시간 처리 성능
>    *  GPU 없이도 CPU 기반 실시간 추론이 가능하다.
>
> * 파이프라인 중심 구조
>    * 단일 모델이 아닌, 전처리 → 추론 → 후처리를 포함한 전체 그래프 구조를 제공
>      
> * 멀티 플랫폼 지원  
>     *  Desktop, Mobile(Android/iOS), Jetson 등 다양한 환경에서 사용 가능
>       
> * 사전 학습된 태스크 제공
>     *  FaceMesh, Hands, Pose, Holistic 등 고수준 태스크를 바로 사용할 수 있음
> ### 3. MediaPipe FaceMesh
>
> * FaceMesh
>   *  MediaPipe FaceMesh는 단일 RGB 이미지에서 사람 얼굴의 468개 랜드마크(landmark) 포인트를 추정하는 모델
>   *  FaceMesh는 단순한 얼굴 검출(face detection)이 아니라, 얼굴의 세부 구조를 정밀하게 추정하는 Face Geometry Estimation에 초점을 둠
>
> * 제공 정보
>    * 얼굴 윤곽
>    * 눈/눈꺼풀
>    * 홍채(iris)
>    * 코
>    * 입술
>    * 턱선
>
